简介

Scrapy

——基于Scrapy 2.4文档。

Scrapy是一个快速的高级网络爬取（web crawling，爬取网站和网站的链接）和网络获取（web scraping，获取网站内容，提取数据）框架，用于爬取网站并且从网站页面提取结构化的数据。用途广泛，可以用于数据挖掘（data mining）、监控和自动化测试等。
